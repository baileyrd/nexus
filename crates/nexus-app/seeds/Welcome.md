# Welcome to Forge

Forge is a **microkernel editor shell** for your notes, code, and
thinking. The chrome you see — the file tree, the inspector, the
status bar — is a thin core; every panel, command, and language
feature rides on top as a plugin.

> Open [[Quick start]] for the short tour, or [[Architecture]] for
> the long one.

## What's here

- `Ideas/` — running lists: [[Tasks]], [[Reading list]].
- `Design/` — longer-form writing: [[Architecture]], [[Decision log]].
- Everything else you write lands next to these; right-click the
  tree for `New note` / `New folder` / `Rename`.

## Getting around

- **⌘K** (or **Ctrl+K**) — command palette. Type to jump to any
  file, switch layouts, toggle themes.
- **Activity rail** (far left) — swap between Files, Search,
  Bookmarks, Tags, Graph.
- **Inspector** (right) — Outline and Backlinks for the doc you're
  reading.

That's it. Delete this note whenever you want; the forge is yours.
