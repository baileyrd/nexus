## Shortcuts worth learning

| Shortcut     | What it does                          |
| ------------ | -------------------------------------- |
| ⌘K           | Command palette                        |
| ⌘P           | File finder (palette scoped to files)  |
| ⌘B           | Toggle file tree                       |
| ⌘J           | Toggle Inspector                       |
| ⌘\           | Focus the editor                       |
