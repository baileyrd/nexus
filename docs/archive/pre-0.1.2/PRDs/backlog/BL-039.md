# BL-039: AI-DIR auto-link suggestions

**Status:** closed ‚Äî 2026-04-29

**Delivered:** AI-INTEGRATION-DIRECTIONS direction 2 lands as a default-off shell-side plugin pair: a CodeMirror extension at [`shell/src/plugins/nexus/editor/cm/linkSuggest.ts`](../../shell/src/plugins/nexus/editor/cm/linkSuggest.ts) that surfaces a `[[wiki-link]]` ghost suggestion at the caret when the typed phrase semantically matches an existing forge note, plus a thin activation plugin at [`shell/src/plugins/nexus/linkSuggest/index.ts`](../../shell/src/plugins/nexus/linkSuggest/index.ts) that owns the configuration schema. **Loop**: `ViewPlugin` debounces 600 ms on each user keystroke (cancel-on-edit semantics mirror BL-034 ‚Äî the `inFlightRequestId` counter is bumped on every doc change so any late resolution falls stale and is discarded). On flush, `extractPhrase(doc, caret, minChars=4, maxChars=80)` walks back from the caret to the nearest hard boundary (`\n` always wins; `. ` / `? ` / `! ` count when followed by a space; otherwise the `caret ‚àí maxChars` window edge). Leading bullet/blockquote/heading markers (`- `, `> `, `* `, `+ `, `# `) are trimmed so a phrase like `"- project atlas"` extracts as `"project atlas"`. The phrase is rejected when shorter than `minChars`, when it contains no letters, or when the caret sits mid-word (the static `\w` lookahead at `text[caret]`). `isInSkipZone(doc, caret)` then suppresses three contexts: an open `[[‚Ä¶]]` (most recent `[[` before the caret with no intervening `]]`); YAML frontmatter (file starts with `---\n` and the caret is before the closing `---` fence ‚Äî handles unclosed frontmatter as "everything is fm"); and fenced code blocks (odd count of `^```$` lines before the caret). **Retrieval**: surviving phrases call `getGhostApi().kernel.invoke('com.nexus.ai', 'semantic_search', { query, limit: 3 })` ‚Äî same handle BL-034 uses, populated by the AI plugin's `setGhostApi(api)`. The top match is gated on `score >= 0.55` (constant `ai.linkSuggest.scoreGate`, tunable). **Two-ghost coexistence**: before the round trip, the fetcher reads `view.state.field(ghostInternals.suggestionField, false)` and DEFERs entirely when BL-034's inline AI ghost is currently visible ‚Äî two ghosts with conflicting Tab semantics would confuse the user, so the cheaper rule is "AI ghost wins, link-suggest waits for the next keystroke after dismissal." `linkSuggest` exposes its own `suggestionField` / decoration / `Prec.high` Tab+Esc keymap, fully isolated from `ghostCompletion`. **Replacement build**: `basenameNoExt(file_path)` strips the dir + `.md`; `buildReplacement(file_path, phrase)` returns `[[basename]]` when `basename.toLowerCase() === phrase.toLowerCase()` (the alias would be noise) and `[[basename|phrase]]` otherwise. **Render**: a single `Decoration.widget` after the caret renders ` ‚Üí [[‚Ä¶]]` in a slightly-bluer muted tone (`var(--accent-muted, #6b8aff)`) so users can distinguish it visually from BL-034's grey AI ghost. Tab accepts and dispatches `{ from: sug.from, to: sug.to, insert: replacement }` with the caret moved to `from + replacement.length`, then clears the field; Esc clears the field. Both bindings are `Prec.high` so they beat the editor's normal Tab/Esc when a suggestion is live and `false`-fall-through when it is not. **Configuration** ([`linkSuggest/index.ts`](../../shell/src/plugins/nexus/linkSuggest/index.ts)) registers five keys under `nexus.linkSuggest` (order 51): `ai.linkSuggest.{enabled,debounceMs,minChars,maxChars,scoreGate}`, all read lazily on each tick via `configStore.get` so live edits in Settings take effect without remounting the editor. **Catalog wiring** ([`shell/src/plugins/catalog.ts`](../../shell/src/plugins/catalog.ts)): registered in `DEFAULT_OFF_PLUGINS` immediately after `semanticSearchPlugin`, with the same comment rationale (depends on the AI plugin's embedding provider being configured ‚Äî pairing the enabled-states keeps users from seeing dead suggestions). **Editor wiring** list (live-mode inherits via the `[...base, livePreviewExt()]` spread). **Coverage**: 23 new tests in [`linkSuggest.test.ts`](../../shell/src/plugins/nexus/editor/cm/linkSuggest.test.ts) re-exported
