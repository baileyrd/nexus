## Pattern observations

- **Editing-experience plugins** dominate: Templater, Advanced Tables, Editing Toolbar, Outliner, Highlightr, Various Complements, Latex Suite, Emoji Toolbar, Linter, Hover Editor.
- **Sync / version control** — Git, Remotely Save, Self-hosted LiveSync, Advanced URI — four entries totalling ~5.6M downloads.
- **AI assistants** — Copilot, Smart Connections, Text Generator. Nexus's built-in `nexus.ai` covers this category natively.
- **Drawing / whiteboards** — Excalidraw, Mind Map, Markmind, Advanced Canvas (four entries).
- **Task / planning** — Tasks, Day Planner, Periodic Notes, Spaced Repetition (four entries).
- **PDF / annotation** — Annotator, PDF++, Zotero Integration (three entries).
- **Themes / chrome customisation** — Style Settings, Iconize, Minimal Theme Settings, Highlightr, Emoji Toolbar (five entries).
